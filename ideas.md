<idea>
  **Design Movement**: "Neural Cyber-Minimalism"
  **Core Principles**:
  1. **Inteligência Visível**: O design deve parecer "vivo" e processando dados, com micro-interações que simulam pensamento.
  2. **Profundidade Infinita**: Uso de camadas escuras sobrepostas para criar uma sensação de imersão em um sistema complexo.
  3. **Precisão Cirúrgica**: Layouts milimetricamente alinhados, transmitindo a ideia de um diagnóstico exato (o core da consultoria).
  4. **Foco no Essencial**: Eliminação de qualquer ruído visual que não contribua para a conversão ou autoridade.

  **Color Philosophy**:
  - **Fundo**: `oklch(0.12 0.01 285)` (Deep Void) - Um preto não absoluto, mas profundo, para reduzir fadiga ocular e destacar o neon.
  - **Primária**: `oklch(0.75 0.18 190)` (Cyan Neural) - Para ações de "avanço" e tecnologia.
  - **Secundária**: `oklch(0.65 0.22 290)` (Purple Insight) - Para elementos de "sabedoria" e transformação.
  - **Texto**: Branco puro para títulos, cinza azulado para corpo, garantindo legibilidade máxima no dark mode.
  - **Intenção**: Criar um ambiente que pareça um "centro de comando" de uma IA avançada.

  **Layout Paradigm**: "Bento Grid Desconstruído".
  - Em vez de seções lineares tradicionais, o conteúdo é organizado em "módulos de dados" que se encaixam, mas com assimetria proposital para guiar o olhar.
  - Uso de linhas finas (1px) com gradientes sutis para separar seções, lembrando placas de circuito ou interfaces de HUD.

  **Signature Elements**:
  1. **"Pulse Nodes"**: Pontos de luz pulsantes nas interseções de linhas ou cantos de cartões, sugerindo atividade.
  2. **"Data Streams"**: Texturas de fundo sutis com código ou dados fluindo verticalmente (muito baixa opacidade).
  3. **"Glass HUD"**: Cartões com fundo translúcido e bordas brilhantes, como displays holográficos.

  **Interaction Philosophy**: "Feedback Háptico Visual".
  - Cada hover deve parecer que o usuário está "ativando" um sistema. Bordas acendem, fundos mudam sutilmente de cor.
  - Scroll suave com elementos que entram em cena com um leve delay (staggered), como se estivessem sendo carregados/processados.

  **Animation**:
  - **Entrada**: Elementos "deslizam e estabilizam" (ease-out-back), como peças de uma máquina se encaixando.
  - **Scroll**: Parallax sutil em elementos de fundo vs. conteúdo.
  - **Destaque**: Textos importantes (como "1 Ajuste") podem ter um efeito de "glitch" ou "digitação" sutil ao aparecer.

  **Typography System**:
  - **Display**: 'Space Grotesk' ou 'Orbitron' (apenas para números grandes ou palavras-chave muito curtas) para o toque futurista.
  - **Headlines**: 'Inter' (Bold/Black) com tracking levemente negativo (-0.02em) para solidez e modernidade.
  - **Body**: 'Inter' ou 'Roboto Mono' (para dados técnicos) para legibilidade técnica e clareza.
</idea>

  **Design Movement**: "Ethereal Bioluminescence"
  **Core Principles**:
  1. **Fluidez Orgânica**: A IA não como máquina fria, mas como uma extensão orgânica da mente. Formas arredondadas e fluidas.
  2. **Luz como Guia**: A luz (neon) não é apenas decoração, ela guia o olhar do usuário para os pontos de conversão.
  3. **Imersão Atmosférica**: Uso pesado de blurs e gradientes radiais para criar uma atmosfera envolvente e misteriosa.
  4. **Clareza Cristalina**: Conteúdo flutuando sobre o fundo, com alto contraste e legibilidade perfeita.

  **Color Philosophy**:
  - **Fundo**: Gradientes profundos de azul meia-noite `oklch(0.15 0.05 260)` a roxo profundo `oklch(0.15 0.08 290)`.
  - **Acentos**: Ciano elétrico `oklch(0.85 0.15 180)` e Magenta vibrante `oklch(0.70 0.20 330)`.
  - **Intenção**: Evocar a sensação de estar dentro de uma rede neural biológica ou um universo digital vasto.

  **Layout Paradigm**: "Floating Z-Axis".
  - Elementos centralizados com forte noção de profundidade (Z-index).
  - O conteúdo principal está sempre "mais perto" do usuário, enquanto elementos decorativos flutuam ao fundo em diferentes planos.

  **Signature Elements**:
  1. **"Glow Orbs"**: Esferas de luz difusa que se movem lentamente ao fundo, criando um ambiente vivo.
  2. **"Frosted Glass"**: Cartões com backdrop-filter: blur() intenso, bordas brancas semitransparentes.
  3. **"Neon Typography"**: Títulos principais com leve efeito de brilho externo (text-shadow) para simular neon real.

  **Interaction Philosophy**: "Levitação Magnética".
  - Botões e cartões parecem flutuar e são atraídos magneticamente pelo cursor.
  - Efeitos de "spotlight" no hover dos cards, revelando bordas ou texturas ocultas.

  **Animation**:
  - **Ambiente**: Animações de loop contínuo e lento para os elementos de fundo (respiração).
  - **Transições**: Fades longos e suaves. Nada é brusco.
  - **Scroll**: Elementos emergem da escuridão (fade-in + translate-y).

  **Typography System**:
  - **Headlines**: 'Montserrat' (ExtraBold) em caixa alta para impacto e geometria.
  - **Body**: 'Lato' ou 'Mulish' para uma leitura suave e amigável.
</idea>

<idea>
  **Design Movement**: "Glitch Brutalism 2.0"
  **Core Principles**:
  1. **Verdade Crua**: Refletindo a promessa da consultoria ("a verdade brutal"), o design é direto, sem enfeites desnecessários, mas esteticamente impactante.
  2. **Contraste Extremo**: Preto total vs. Cores Neon saturadas. Sem tons de cinza intermediários suaves.
  3. **Tipografia Gigante**: O texto é a imagem. Headlines ocupam 50% da tela.
  4. **Estética de Terminal**: Referências a código, terminais de comando e logs de sistema, reforçando a natureza "AI-Driven".

  **Color Philosophy**:
  - **Fundo**: Preto Absoluto (#000000).
  - **Primária**: Verde Terminal `oklch(0.7 0.2 140)` (referência a sucesso/código validado).
  - **Secundária**: Roxo Digital `oklch(0.6 0.25 300)` (para elementos de IA/Futuro).
  - **Alerta**: Amarelo Código `oklch(0.85 0.18 95)` (para avisos e escassez).
  - **Intenção**: Transmitir urgência, tecnologia bruta e resultados diretos.

  **Layout Paradigm**: "Modular Grid System".
  - Grades visíveis, linhas divisórias espessas.
  - Layouts assimétricos inspirados em revistas de design suíço, mas digitalizados.

  **Signature Elements**:
  1. **"Raw Borders"**: Bordas de 2px ou 3px sólidas, sem arredondamento (border-radius: 0).
  2. **"Monospace Accents"**: Uso de fontes monoespaçadas para metadados, datas e etiquetas.
  3. **"Glitch Effects"**: Imagens e textos com efeitos de distorção cromática (RGB split) no hover ou scroll rápido.

  **Interaction Philosophy**: "Instantâneo e Mecânico".
  - Hover states são binários: invertem cores (preto vira verde, verde vira preto) instantaneamente, sem transição suave.
  - Botões reagem ao clique com deslocamento físico (box-shadow sólido).

  **Animation**:
  - **Entrada**: Cortes secos ou efeitos de "reveal" tipo persiana.
  - **Texto**: Efeito de "typewriter" (datilografia) para headlines.
  - **Scroll**: Sticky headers e elementos que se fixam enquanto outros passam.

  **Typography System**:
  - **Headlines**: 'Sora' ou 'Chakra Petch' (para um visual mais "tech/gamer").
  - **Body**: 'JetBrains Mono' ou 'Fira Code' (legibilidade de código, estética dev).
</idea>
